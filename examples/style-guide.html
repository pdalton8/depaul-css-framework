Getting Started

Basic HTML Template

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Page Title</title>
    
    <!-- Google Fonts (Required) -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Rethink+Sans:wght@300;400;500;600;700;800&family=Newsreader:ital,wght@0,400;0,500;0,600;0,700;1,400&display=swap" rel="stylesheet">
    
    <!-- DePaul Bootstrap Theme -->
    <link rel="stylesheet" href="css/depaul-bootstrap-theme.css">
    
    <!-- Bootstrap JS (Required for navbar collapse only) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" defer></script>
</head>
<body>
    <!-- Your content here -->
</body>
</html>

Design Specifications

  • Border Radius: Asymmetric - top-right and bottom-left corners rounded (1rem)
  • Font Size: 1.125rem (18px)
  • Font Weight: 700 (Bold)
  • Min Width: 13.25rem
  • Transition: 0.5s cubic-bezier(0.19, 1, 0.22, 1)
  • Arrow Animation: Slides 0.375rem on hover

                </li>
            </ul>
        </div>
    </div>
</nav>

Navigation Features

  • Background: DePaul Navy (#00205b)
  • Collapse: Uses Bootstrap's collapse.js for mobile menu
  • Sticky: Can be made sticky with position: sticky
  • Mobile: Hamburger icon transforms to X when open
  • Alignment: Logo left, navigation right with .ms-auto

Spotlight Accordion

Interactive horizontal accordion with expandable panels. Click on any panel to expand it. Includes JavaScript for interaction.

  • Layered-Learning™

    Our unique approach combines classroom lessons with project-based and community-based service learning.

  • Classroom Learning

    With a 17:1 student-to-faculty ratio, classroom learning is not a distant lecture but a living dialogue.

  • Project Based Learning

    We challenge students to learn by doing, standing shoulder to shoulder with experienced faculty.

Note: This component requires JavaScript to function. Add this script to your page:

<script>
document.addEventListener('DOMContentLoaded', function() {
    const items = document.querySelectorAll('.spotlight-accordion__item');
    items.forEach(item => {
        item.addEventListener('click', function() {
            items.forEach(i => i.classList.remove('active'));
            this.classList.add('active');
        });
    });
});
</script>
<section class="spotlight-accordion" style="--total-slides: 5">
    <ul class="spotlight-accordion__list">
        <li class="spotlight-accordion__item active">
            <div class="spotlight-accordion__item-bg">
                <img src="image.jpg" alt="Alt text">
            </div>
            <div class="spotlight-accordion__content">
                <h4 class="spotlight-accordion__title">Title</h4>
                <p class="spotlight-accordion__description">Description</p>
                <button class="btn btn-secondary btn-arrow">Button</button>
            </div>
        </li>
        <!-- Repeat for multiple items -->
    </ul>
</section>
